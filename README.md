# Zomato Bangalore Restaurant Analysis

## Project Overview
Analyzed 8,921 Zomato restaurants in Bangalore to identify 
key insights on ratings, cuisine preferences, area-wise 
distribution and cost analysis.

## Tools Used
- Microsoft Excel (Data Cleaning & Analysis)
- SQL (Query Writing & Analysis)
- Python (Data Visualization — Matplotlib & Seaborn)
- Power BI (Interactive Dashboard)

## Key Insights
- Electronic City has most restaurants (674)
- 92.8% restaurants are Non-Veg
- Average cost for two is ₹340
- Indiranagar is most premium area (₹479 avg cost)
- Average delivery rating is 3.88 across Bangalore
- Veg Only restaurants rated higher (4.0) vs Non-Veg (3.87)
